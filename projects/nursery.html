Outback Nursery Management System

   

Web Application Development | E-commerce UI/UX | MySQL

 
   
   

Overview & Project Goal

   

The **Outback Nursery Management Project** is a web-based application designed to streamline traditional nursery operations by efficiently managing inventory and customer data. Developed as a coursework initiative, the system showcases the practical application of modern web development technologies to digitize retail processes.

   

My Role & Technologies

   
     
       
             
  • **Role:** Full-Stack Developer & UI Designer
    •          
  • **Focus:** Data Structure, E-commerce User Flow, Responsive Design
    •        
     
     
       
             
  • **Database:** MySQL (for robust data handling)
    •          
  • **Front-end:** HTML/CSS, Bootstrap (for responsive design)
    •          
  • **Key Feature:** Separate Admin/User roles
    •        
     
   
   
       

Clean & Intuitive User Interface

   

The front-end design uses **Bootstrap** for responsiveness, ensuring the site looks great on mobile, tablet, and desktop. The visual style uses natural greens and white space to create a welcoming, plant-focused atmosphere.

   
                 

The main landing page, focusing on high-quality visuals and clear CTAs.

   
       

System Functionality: Admin & User Roles

   

The platform features secure, separate user roles to manage internal operations and external customer engagement:

       
     
       

Admin Panel Functionality

       

Admins can perform **CRUD (Create, Read, Update, Delete)** operations on products, categories, and inventory through a secure, dedicated admin panel. This streamlines stock management and product listing updates.

     
     
       

Customer Features

       

Users can create accounts, easily browse plants, view product details, and save items to a **Favorites** list, enhancing their shopping experience and encouraging repeat visits.

     
   
       

Product Navigation & Detail

   

Effective organization was key. Products are easily filtered via categories, and each product page provides essential details, pricing, and clear inventory status.

       
     
                       

Browsing plants via distinct categories (e.g., Indoor, Outdoor)
