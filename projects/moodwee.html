Moodwee – Mood Tracking App

UI/UX • WordPress • Custom Plugin

Overview

Moodwee is a mood tracking web app designed to help users monitor emotional wellness. I created the UI design in soft, calming tones and built a custom WordPress plugin to store moods in a database and visualize them with charts.

My Role

  • UI/UX Design in Figma
  • Front-end development with HTML, CSS, WordPress Elementor
  • Custom plugin development for mood tracking

Process

I started with user research → wireframes → UI mockups → plugin development. The goal was to keep the interface simple and motivational.

Results

- Created a functional app with personalized wellness tips & reminders.
- Integrated graphs & downloadable reports.
- Received positive feedback on clean UI and usability.
